## Para rodar o projeto localmente:
### 1. Clonar o repositório
```bash
git clone https://github.com/danielrpiccoli/trabalho-icn-2025.git
cd trabalho-icn-2025
```
### 2. Criar e ativar o ambiente virtual (para instalar as dependências isoladamente)
No Linux:
```bash
python3 -m venv .venv
source .venv/bin/activate
```
No Windows: 
```powershell
python -m venv .venv
.venv\Scripts\Activate
```
### 3. Instalar dependências
```bash
pip install -r requirements.txt
```
### 4. Executar notebook
```bash
jupyter notebook
```
